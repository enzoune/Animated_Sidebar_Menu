# 📂 Sidebar Menu System (Barra Lateral Interativa)

> Um componente de navegação lateral (Sidebar) profissional e responsivo, projetado para dashboards e sistemas administrativos, utilizando lógica declarativa em CSS3.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir uma **Barra Lateral de Menu** que equilibre estética moderna com funcionalidade técnica rigorosa. 

O componente apresenta uma lista de navegação completa, incluindo links para Dashboard, Relatórios, Configurações e Mídias Sociais, garantindo que o usuário tenha todos os controles da aplicação ao alcance de um clique.

## ✨ Destaques e Diferenciais Técnicos

* **Gerenciamento de Estado Nativo (No-JS):** Implementação da técnica "Checkbox Hack" (`input type="checkbox" id="check"`) para controlar a abertura e fechamento do menu puramente via CSS, garantindo máxima performance.
* **Integração Multi-Biblioteca:** Uso estratégico de duas bibliotecas de ícones distintas — **Boxicons** para elementos de branding e interface, e **Material Symbols Outlined** da Google para itens de menu — demonstrando versatilidade na integração de recursos externos.
* **Arquitetura de UI Modular:** Estruturação através de classes bem definidas como `.sidebar_menu`, `.menu` e `.social_media`, o que facilita a manutenção e a escalabilidade do código.
* **UX Refinada:** Inclusão de botões de alternância dinâmicos (ícones de menu e fechar) vinculados à label do checkbox, proporcionando uma interação tátil e intuitiva.
* **Organização Profissional:** Separação clara entre a estrutura semântica (`index.html`) e a identidade visual (`css/style.css`), seguindo os padrões recomendados de engenharia de software.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação e lógica de estado nativa.
* **CSS3:** Estilização avançada, gerenciamento de transições e design responsivo.
* **Boxicons:** Iconografia de interface e redes sociais.
* **Google Material Symbols:** Iconografia técnica para itens de dashboard.
